IDS y Clases en CSS (Cascading Style Sheets)

Para identificar un párrafo con CSS sin que todos los párrafos que hayan, es necesario hacer uso del atributo Class para elementos específicos y, el atributo id comparta una misma función, al igual que éstos son atributos globales es decir, que se pueden usar en cualquier elemento HTML. en CSS para acceder a elementos id o Class se usa (*) para ID y (.) para Class

Si utilizamos un ID (Atributo Global) con un nombre específico, no se va a poder utilizar ese mismo ID con otros elemntos. Para el atributo Class se puede utilizar varias clases dentro de ese misma etiqueta separadas por espacio para que lo logre identificar con clases distintas.
