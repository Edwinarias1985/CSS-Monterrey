Agregar CSS (Cascading Style Sheets) a HTML

Se evidencia en agregar CSS a HTML, primero desde la etiqueta Link dentro de la etiqueta Head , la segunda forma es utilizando la etiqueta Style dentro del head agregando un selector en este caso el h1 para estilarlo. Y la tercera forma es utilizando Style que es un atributo global en cualquier etiqueta en este caso lo usé en la etiqueta p.
