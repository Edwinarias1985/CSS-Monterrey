Introducción a CSS (Cascading Style Cheets)

CSS se trata de, decorar la página WEB. Es un tipo de documento que nos ayuda a añadir estilos a nuestros elementos HTML, tales como color, letra, color de letra, tamaños etc

Las competencias adquiridas son:

  • Conocer lo básico sobre el lenguaje de estilo CSS
  • Identificar tamaños y responsividad adecuados
  • Integrar diferentes fuentes de texto, colores y rellenos.
  • Modificar el posicionamiento de elementos.
