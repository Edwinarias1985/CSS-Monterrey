Recordemos que en el módulo anterior se aprendió sobre un elemento HTML cuya función es permitir la modificación de pequeñas partes del documento, como ejemplo se le puede poner a una palabra, letra(s), párrafo el color que queramos par que resalte. Las Divisiones son elementos que con funciones similares a un spanpero la diferencia es que las Divisiones se usan para agrupar bloques enteros en lugar de frases o palabras. Para poder hacer uso de estas Divisiones se debe de emplear la etiqueta div, normalmente están acompañadas de una calse o ID para poder modificar el contenido dentro sin afectar otros elementos que estén alrededor.

Herencia en CSS (Cascading Style Sheets)

Se puende observar que el color del txto en el segundo contenedor cambia: en la primer ejemplo es negro y el segundo blanco. Esto se da porque en el segundo ejemplo la segúnda división está dentro de la primera. Locual causa que se hereden las propiedades de la primera división a la segúnda. El CSS de la primera división contiene color: white; y es esta propiedad la que se hereda a la segunda división, cambiando el color del texto. La otra propiedad que se hereda es text-align: center;, la cual centra al texto en su contenedor.
A esto se le denomina herencia, la cual consiste en aquellas propiedades que se pasan de un elemnto padre o superior a otro elemnto hijo. El color negro que se asigna por defecto al texto del documento; al modificar el contenedor con un nuevo color, este se aplica a todos los elementos dentro de ese contenedor. Si agregamos la propiedad color al segundo contenedor, entonces se sobreescribe la propiedad heredada de la primer división. Es por esto que el tipo de letra del primer contenedor no se aplica al segundo, ya que tenemos otra propiedad font-family siendo modificada.
