Tema 4: atajos de clases e IDs.

  • En VS Code nos facilita agragar clases e IDs a elementos nuevos, para esto podemos escribir el nombre del tipo de elemento seguido de un "." o un "#" y seguio el nombre de la clase.

    • p.ejmeplo.segundo-ejemplo

  • Podemos agregar múltiples clases al elemento iniciando cada nombre por un ".".

    • jlajsdlfajlsdfjl

A diferencia de las clases, no pordemos agragar más de un ID a un elemento

  • Usando Clases

    • h1.primer-titulo.otra-clase

  • Usando ID's

    • h1#unico#diferente

  • Usando SUB etiquetas

    • div>h2+p+ul>li*3

  • Usando el atajo para hacer el div

    • .clase>h1+p

  • Usando corchetes planos para ATRIBUTOS

    • h1.clase.clase-dos[style="background-color: purple;"title="Soy un h1"]

  • Usando llaves para contenido de la etiqueta

    • h1.clase.clase-dos[style="color: yellow;"title="Soy un h1"]{Hello honey}
